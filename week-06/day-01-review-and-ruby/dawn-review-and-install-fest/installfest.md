#Installation party!

0.  Update Brew
	`brew update`

0.  Install Postgress
	`brew install postgres`

0.  Install Rails
	`gem install rails`

1.  Install GPG to install RVM

	`brew install gpg`

2. Install RVM

	`gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3`

3.  Install RSPEC
	`gem install rspec`
4. Install gem bundler
	`gem install bundler`
